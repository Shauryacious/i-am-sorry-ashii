# for ashii 🥺

a tiny mobile-first apology in six scenes:

1. **the door** — tap to open
2. **her name falling like rain** + a broken heart
3. **the runaway "no"** button (you can only say yes)
4. **swipeable apology cards** — tinder for sorrys
5. **a handwritten letter** that types itself
6. **the heart reassembles** + a forgive button + petal burst

pure HTML/CSS/JS. no build step. mobile-first, haptic, ambient petals on a canvas.

## run locally
```bash
npx serve .
```

## deploy
push to GitHub → import to Vercel → done. or `npx vercel`.
